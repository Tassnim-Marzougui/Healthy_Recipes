Healthy_Recipes est une application web développée avec .NET, dédiée à la promotion d’un mode de vie sain à travers des recettes équilibrées et personnalisées. L’application intègre un chatbot intelligent orienté santé qui guide les utilisateurs dans le choix de recettes adaptées à leurs besoins (alimentation saine, objectifs nutritionnels, habitudes alimentaires). Grâce à une interface moderne et conviviale, les utilisateurs peuvent découvrir des recettes healthy, recevoir des conseils nutritionnels et interagir avec le chatbot pour obtenir des recommandations simples et utiles. Ce projet vise à allier technologie, bien-être et intelligence artificielle pour encourager de meilleures habitudes alimentaires
